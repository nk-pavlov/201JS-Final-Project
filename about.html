About the project

Built to learn.
Built to ship.

ShopFlow is a JavaScript final project — a fully client-side product catalog dashboard. No frameworks, no build tools, just vanilla JS, ES6 modules, and the Fetch API.

The goal: cover every core JS concept — async/await, array methods, DOM manipulation, event delegation, localStorage, and modular code structure.

Features Implemented

✓Protected route — dashboard redirects to login if no user in localStorage
✓Async data fetching — products loaded from FakeStore API with loader/error states
✓Search & filter — real-time search + category filter combined
✓Sorting — price asc/desc, name A→Z and Z→A
✓Add product — modal form, validation, localStorage persistence
✓Delete product — event delegation on grid, updates localStorage
✓Cart drawer — add/remove items, total with Array.reduce()
✓Stats row — total, categories via Set(), avg price via reduce()
